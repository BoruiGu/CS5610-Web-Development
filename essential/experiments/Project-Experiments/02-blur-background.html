Week 8 - Blurred Background

In this expriment, I apply CSS blur filter to create a blurred background, which will be used in the homepage of my project.

<div class="content">
    <div class="bg-img"></div>
    <p>Hello World</p>
</div>

A seperate div is created for the background image since blur effect will apply to the whole element.

.bg-img {    
    background: url(http://images.all-free-download.com/images/graphiclarge/black_wall_266397.jpg) no-repeat center center fixed;
    background-size: cover;
    width: 100%;
    height: 480px;
    -webkit-filter: blur(7px);
    -moz-filter: blur(7px);
    -o-filter: blur(7px);
    filter: blur(7px);
}
.content {
    color: #efefef;
    position: relative;
}
.content p {
    position: absolute;
    top: 0px;
    width: 100%;
    font-size: 40px;
    text-align: center;
    vertical-align: middle;
    line-height: 480px;
}

Note that the parent element is relatively positioned, thus the absolutely positioned child element can be placed relative to its parent.
