Week 9 - Player - Change Progress & Volume

This experiment is based on a previous expeirment.
User can drag progress bar to change current progress of the audio.
Note: When playerStarted is false (i.e. either the audio has not been loaded, or the audio has reached the end), the audio will always play from the beginning when "Play" is clicked.
A vertical volume bar is added. User can drag it to change volume.

Progress: {{progress}}ms

Duration: {{duration}}ms

Volume: {{volume}}%

$scope.changeProgress = function () {
    player.currentTime = $scope.progress / 1000.0;
};
$scope.changeVolume = function () {
    player.volume = $scope.volume / 100.0;
};
