Week 8 - Custom Scroll Bar

In this experiment, I create customized scroll bar styling to match my project's theme.

::-webkit-scrollbar {
    background-color: #373737;
    width: 7px;
}
::-webkit-scrollbar-track {
    background-color: #373737;
}
::-webkit-scrollbar-thumb {
    background-color: #676769;
}
::-webkit-scrollbar-thumb:active {
    background-color: #919191;
}
::-webkit-scrollbar-corner {
  background-color: #373737;
}
body {
    background-color: #121314;
    color: #DFD3B4;
}
Note: The scrollbar may look different in Chrome and IE.
