In this experiment array is used to fill in contents of a list.

Method1: Iterate through <li> tags in the list and replace content with content we want.

  • placeholder 1
  • placeholder 2
  • placeholder 3
var list = $("ul#worklist");
var items = ["Meet", "Your", "Makers"];
list.find("li").each(function (index) {
    console.log($(this).text());
    $(this).html(items[index]);
})

Method2: First clear the list, then iterate through items in the array, create <li> tags, and append to list.

  • placeholder 1
  • placeholder 2
  • placeholder 3
list = $("ul#worklist1");
list.empty();
for (var i in items) {
    var li = $("<li>" + items[i] + "</li>");
    list.append(li);
}
