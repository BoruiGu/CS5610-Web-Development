The two buttons above control show / hide of the two paragraphs below them. Also text in the buttons will change accordingly.

HTML source: 

<button id="btnP1">
    <p class="btnTextShow">Click me to show paragraph 1</p>
    <p class="btnTextHide">Click me to hide paragraph 1</p>
</button>
<button id="btnP2">
    <p class="btnTextShow">Click me to show paragraph 2</p>
    <p class="btnTextHide">Click me to hide paragraph 2</p>
</button>
<p id="p1"><b>Paragraph 1</b></p>
<p id="p2"><b>Paragraph 2</b></p>

jQuery source: 

var p1 = $("#p1");
var p2 = $("#p2");
p1.hide();    
p2.hide();
$(".btnTextHide").hide();
function btnControl(btn, p) {
    var btnTextShow = btn.find(".btnTextShow");
    var btnTextHide = btn.find(".btnTextHide");
    btn.toggle(
    function () {
        p.show();
        btnTextShow.hide();
        btnTextHide.show();
    },
    function () {
        p.hide();
        btnTextShow.show();
        btnTextHide.hide();
    });
}
btnControl($("#btnP1"), p1);
btnControl($("#btnP2"), p2);

Note that since control functions of these two buttons are similar, a generalized function is created. A button and the paragraph it controls are passed to this generalized function.

Also, prepand() and append() are used to add content to paragraph.

$("[id^='p']").append(" This is appended to the paragraph.");
$("[id^='p']").prepend(" This is prepended to the paragraph.");
