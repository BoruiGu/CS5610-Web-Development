Week4 - WYSIWYG Editor

Here I'm trying to create a WYSIWYG editor using HTML5's contenteditable attribute.
Bold / italic / underline format can be added to words by pressing CTRL+B / CTRL+L / CTRL+U, or by clicking the buttons.

Try editing this content.

In html: 
<p id="editor" contenteditable="true">Try editing this content.</p>
In javascript: 
$("#bold").click(function () {
    document.execCommand("Bold", false, null);
});
$("#italic").click(function () {
    document.execCommand("Italic", false, null);
});
$("#underline").click(function () {
    document.execCommand("Underline", false, null);
});
