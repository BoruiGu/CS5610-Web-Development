Week3 - Selector

This experiment focuses on selecting elements in the DOM and changing their style.

Select all elements with a class name containing "description" and change font-style to "italic"

$("[class*='description']").css("font-style", "italic");

all texts in class "uppercase" would be displayed in uppercase.

$("p.uppercase").css({
    "text-transform": "uppercase",
    "color": "firebrick"
});

select a paragraph by its id and change its style.

$("p#bgColor").css("background-color", "grey");

Select all odd <p> tags and change background color.

$("p:odd").css({
    "color": "white",
    "background-color": "black"
});
