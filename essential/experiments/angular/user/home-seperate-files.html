Week6 - Login - Seperate - Files

This experiment is based on a previous experiment.
In this experiment, javascripts are seperated into 4 files:

  • route.js: configuration for $routeProvider
  • navController.js: navController
  • profileController.js: profileController
  • UserService.js: UserService
Global variable app is defined in route.js. In html, link to route.js should be placed before links to other .js files.

html: 
<script src="js/09/route.js"></script>
<script src="js/09/navController.js"></script>
<script src="js/09/profileController.js"></script>
<script src="js/09/UserService.js"></script>
route.js: 
var app = angular.module("UserApp", ['ngRoute']);
app.config(['$routeProvider',
...
}]);
navcontroller.js: 
app.controller("navController", function ($scope, UserService, $location) {
...
});
