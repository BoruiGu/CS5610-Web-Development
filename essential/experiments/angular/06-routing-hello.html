Week5 - Routing - Hello

First experiment in Angular Routing.
The two links below do not link to actual files. Instead we configure $routeProvider, and it loads contents from files and fills into the div marked by "ng-view".

Home Bar
In html: 
<a href="#/home">Home</a>
<a href="#/bar">Bar</a>
<div class="container" ng-view>
</div>
In javascript: 
app.config(['$routeProvider',
function ($routeProvider) {
    $routeProvider.
      when('/home', {
          templateUrl: 'partials/hello-home.html'//,
          //controller: 'PhoneListCtrl'
      }).
      when('/bar/', {
          templateUrl: 'partials/hello-bar.html'//,
          //controller: 'PhoneDetailCtrl'
      }).
      otherwise({
          redirectTo: '/home'
      });
}]);
