Week4 - Array Object

In this experiment I rewrite an experiment Array Object performed last week with AngularJS. ng-repeat is simple and powerful.

# a b c
{{c.c1}} {{c.c2}} {{c.c3}} {{c.c4}}
In html: 
<tbody>
    <tr ng-repeat="c in content">
        <td>{{c.c1}}</td>
        <td>{{c.c2}}</td>
        <td>{{c.c3}}</td>
        <td>{{c.c4}}</td>
    </tr>
</tbody>
In javascript: 
$scope.content = content;
