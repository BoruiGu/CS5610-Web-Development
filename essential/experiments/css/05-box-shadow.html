Week7 - Box Shadow

CSS3 border-radius property enables us to add shadows to elements.
Property Values: Horizontal Length, Vertical Length, Blur Radius, Spread Radius, Shadow Color, Inset(inner shadow).
Elements can have multiple shadows, specified by a comma seperated list. When more than one shadow is specified, the shadows are layered front to back.
CSS Generator: CSSmatic 



<div class="box">
</div>

.box {
    background-color: brown;
    width: 360px;
    height: 360px;
    border-radius: 50px 50px 50px 50px;
    -moz-border-radius: 50px 50px 50px 50px;
    -webkit-border-radius: 50px 50px 50px 50px;
    border: 0px solid #000000;
    -webkit-box-shadow: 0px 0px 15px 7px rgba(173, 104, 59, 1), 
        0px 0px 33px 17px rgba(137, 106, 88, 1);
    -moz-box-shadow: 0px 0px 15px 7px rgba(173, 104, 59, 1), 
        0px 0px 33px 17px rgba(137, 106, 88, 1);
    box-shadow: 0px 0px 15px 7px rgba(173, 104, 59, 1), 
        0px 0px 33px 17px rgba(137, 106, 88, 1);
}
