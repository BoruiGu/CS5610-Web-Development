For desktop:

There is a menu at the top of the page. The space below is divided into three columns. (col-3, col-6, and col-3) The middle column shows most important contents. The left column shows less important contents. The right column contains links to relative sites.

@media screen and (min-width: 1024px) {
    .col-3 {
        float: left;
        width: 238px;
    }
    
    .row-150 {
        height: 150px;
        border: 2px solid lightgrey;
    }
}

For phone:

The menu will now be shown. Contents will be placed vertically, in the order of most important contents - less important contents - relative links.

@media screen and (max-width: 1024px) {
    .main-content nav {
        display: none;
    }
    .main-content section {
        display: table;
    }
    .main-content section #most-important {
        display: table-header-group;
    }
    .main-content section #less-important {
        display: table-row-group;
    }
    .main-content section #relative-links {
        display: table-footer-group;
    }
}
